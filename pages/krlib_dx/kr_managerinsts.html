class ManagerInsts

ManagerBase の派生クラスを、生成・管理します。

使用例

初めに App::InitDx() の前で ManagerBase の派生クラスを生成する必要があります。

生成したクラスは、関数が自動で実行されます。
実行の順番は、 ManagerBase の order値 が少ない順から実行されます。 


              //管理クラスを一斉に取得.
              vector<ManagerBase*> managers = {
                ManagerInsts::GetInst().Get<MyClass1>(),
                ManagerInsts::GetInst().Get<MyClass2>(),
                ManagerInsts::GetInst().Get<MyClass3>()
              };
              //全てループ.
              for (auto& i : managers) {
                i->SetAutoExeMode(MngAutoExe::Stop); //Update, Drawの稼働を停止する.
              }
