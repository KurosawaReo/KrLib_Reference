class ManagerInsts

ManagerBase の派生クラスを、生成・管理します。
生成した管理クラスは、自動で関数が実行されます。

プロパティ
なし
メソッド
NewManager Get GetAll SortOrder
クリック / タップで開く
使用例

初めに App::InitDx() の前で ManagerBase の派生クラスを生成する必要があります。 


              int WINAPI WinMain(
                /* 略 */
              ){
                //Managerクラス実体生成.
                //引数で実行順(order値)を入力する.
                ManagerInsts::NewManager<InputMng>(0);
                ManagerInsts::NewManager<SoundMng>(1);
                ManagerInsts::NewManager<SceneMng>(2);

                ManagerInsts::NewManager<MyClass1>(3);
                ManagerInsts::NewManager<MyClass2>(4);
                ManagerInsts::NewManager<MyClass3>(5);

                /* [略] App::InitDx, App::LoopDx をここに記述 */

                return 0;
              }

以下のように、複数の管理クラスを取得して一斉に処理することも可能です。 


              //管理クラスを一斉に取得.
              vector<ManagerBase*> managers = {
                ManagerInsts::Get<MyClass1>(),
                ManagerInsts::Get<MyClass2>(),
                ManagerInsts::Get<MyClass3>()
              };
              //全てループ.
              for (auto& i : managers) {
                i->SetAutoExeMode(MngAutoExe::Stop); //Update, Drawの稼働を停止する.
              }
