class App

アプリケーションの初期化・ループなどの制御を行うクラスです。

プロパティ
なし
メソッド

使用例

main関数で InitDx LoopDx を実行します。
ManagerBase を継承した管理クラスの Init Reset Update Draw 関数が自動で呼ばれるようになります。 


          int WINAPI WinMain(
            _In_     HINSTANCE hinstance,
            _In_opt_ HINSTANCE hPrevinstance,
            _In_     LPSTR     lpCmdLine,
            _In_     int       nCmdShow
          ){
            //初期化.
            ResultInt err = App::InitDx(800, 600, true, 60, false);
            if (err.GetCode() < 0) { return -1; } //初期化エラー.
            //ループ.
            App::LoopDx();

            return 0;
          }
