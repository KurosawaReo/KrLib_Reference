class ManagerInsts

管理クラスの実体を保存する用のクラスです。
ManagerBase のコンストラクタを実行すると、自動でこちらのクラスに実体が保存されます。

使用例

主に ManagerBase を継承した管理クラスを取得するのに使います。 


              //管理クラスを一斉に取得.
              vector<ManagerBase*> managers = {
                ManagerInsts::GetInst().Get<MyClass1>(),
                ManagerInsts::GetInst().Get<MyClass2>(),
                ManagerInsts::GetInst().Get<MyClass3>()
              };
              //全てループ.
              for (auto& i : managers) {
                i->SetAutoExeMode(MngAutoExe::Stop); //Update, Drawの稼働を停止する.
              }

class ManagerBase

クリック / タップで開く
使用例

コンストラクタでorder値(管理クラスの実行順)をセットする必要があります。
order値が少ないクラスから順に実行されます。 


              class GameManager : public ManagerBase
              {
              private:
              public:
                //コンストラクタ.
                GameManager() : ManagerBase(0) {}
                
                //この4つの関数が自動で実行されるようになる.
                void Init()   override {}
                void Reset()  override {}
                void Update() override {}
                void Draw()   override {}
              };
