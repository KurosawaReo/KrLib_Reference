enum MngAutoExe

管理クラスの自動実行モード。 Update Draw を自動で動かすか動かさないかを設定するのに使います。

Active
どちらも実行
UpdateOnly
Updateのみ実行
DrawOnly
Drawのみ実行
Stop
実行しない
class ManagerBase

管理クラスの機能を持った基底クラスです。
派生したクラスを ManagerInsts で生成すると、4つの関数 Init Reset Update Draw が自動で実行されます。

使用例

コンストラクタで order値 をセットする必要があります。
以下のようにすると ManagerInsts::NewManager() からでもorder値を設定できるようになるのでおすすめです。 


              class GameManager : public ManagerBase
              {
              private:
              public:
                //コンストラクタ(引数でorder値を入力する)
                GameManager(int order) : ManagerBase(order) {}
                
                //この4つの関数が自動で実行されるようになる.
                void Init()   override {}
                void Reset()  override {}
                void Update() override {}
                void Draw()   override {}
              };
